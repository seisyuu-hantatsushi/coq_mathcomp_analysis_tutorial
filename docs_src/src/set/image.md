# 写像

## 写像
### 写像
2つの集合の\\(X,Y\\)の要素\\( \forall x \in X, \forall y \in Y \\)において,\\(x\\)に\\(y\\)を唯一つ対応させる**写像(関数)**\\(f\\)を
\\[
	f: X \mapsto Y
\\]
で定義する.\\(X\\)を**定義域**,\\(Y\\)を**値域**という.
\\[
	y = f(x)
\\]
と書いたとき,\\(y\\)を\\(f\\)による\\(x\\)の**値**という.
\\(A \subset X, B \subset Y\\)のとき
\\[
B = f(A)
\\]
ならば,\\(B\\)を\\(f(A)\\)の**像**という.
